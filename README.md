# Infrastructure
This folder contains Kubernetes (`k8s`) manifests for deploying ScyllaDB, Redis, and Etcd, aswell as scripts for automatic cluster scaling and deployment.

## Tasks:

- #### Making this exectable!!
- Setting up Minikube
- Running the System

Startup with:
```
chmod +x infra/scripts/deploy.sh
```
Run with:
```
./deploy.sh
```
